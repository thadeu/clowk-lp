---
title: List Users
description: List all users in an instance
---

## GET /api/v1/users

Returns all users that belong to the instance associated with your secret key.

## Request

```bash
curl https://myapp.clowk.dev/api/v1/users \
  -H "X-Clowk-Secret-Key: sk_live_..."
```

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `X-Clowk-Secret-Key` | Yes | Your instance secret key |

## Response

### 200 — Success

```json
[
  {
    "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
    "resource": "user",
    "data": {
      "email": "jane@example.com",
      "name": "Jane Doe",
      "avatar_url": "https://lh3.googleusercontent.com/...",
      "email_verified": true,
      "providers": ["google"],
      "created_at": "2026-03-20T10:30:00Z"
    }
  }
]
```

### 401 — Unauthorized

```json
{ "error": "Unauthorized" }
```
