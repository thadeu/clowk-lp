## SDK contributions

Clowk SDKs are open to contributions. If you find a bug or want to add a feature:

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Community SDKs

Want to build a Clowk SDK for a framework we don't support yet? We'd love to help. Reach out at [support@clowk.in](mailto:support@clowk.in) and we'll provide guidance on the API contract and testing.

## Documentation

Found a typo or want to improve the docs? The documentation source lives alongside the landing page. Pull requests are welcome.

## Bug reports

Report bugs at [support@clowk.in](mailto:support@clowk.in) with:

- Steps to reproduce
- Expected behavior
- Actual behavior
- SDK version and runtime

## Feature requests

Email [support@clowk.in](mailto:support@clowk.in) with your use case. We prioritize features based on demand and alignment with Clowk's broker model.
